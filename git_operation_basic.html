## 增加一个新的项目到github ### 在github上 new 一个 resposity ### 在本地 git clone 一个新的文件夹下来 ### 把你要添加的内容放到文件夹下面 ``` git add * git commit git push ``` ## 更新项目内容 如果有新的文件加入，需要重新 git add * 添加一下 普通更新就 ``` git commit git push ```

在本地 git clone 一个新的文件夹下来

把你要添加的内容放到文件夹下面

git add *
git commit
git push
